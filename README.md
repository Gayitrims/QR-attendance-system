# QR-attendance-system
a QR code based attendance management system for automating classroom attendance
## Overview
This project is a smart attendance system that uses QR codes to automate attendance marking in classrooms.

## Features
- QR code generation
- QR code scanning
- Attendance storage
- Student management
- Attendance reports
- Secure login system

## Technologies Used
- HTML
- CSS
- JavaScript
- Python
- MySQL

## Working
1. Admin registers students
2. QR codes are generated
3. Students scan QR codes
4. Attendance is stored automatically
5. Faculty can view reports

## Future Improvements
- Face recognition
- RFID integration
- Mobile app support
- Cloud database
