# QR Attendance Management System

## Introduction
The QR Attendance System is designed to automate attendance marking using QR codes.

## Objective
To reduce manual attendance work and improve attendance management.

## Problem Statement
Traditional attendance systems consume time and allow proxy attendance.

## Proposed Solution
Each student receives a unique QR code.
Students scan the QR code during class attendance.

## Advantages
- Faster attendance
- Easy data management
- Reduced paperwork
- Better accuracy

## Technologies Used
- HTML
- Python
- QR Code Technology

## Future Scope
- Face Recognition
- Mobile Application
- Cloud Storage
